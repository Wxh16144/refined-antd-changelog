# refined-antd-changelog

清理 Ant Design 的 changelog，折叠不推荐使用（不适合生产环境）的版本，计算最佳建议版本。(油猴脚本)

[Install on Greasyfork](https://greasyfork.org/en/scripts/484164-refined-ant-design-changelog)
